## Signal-Only 전용 Stop 조건

> Signal-only는 P&L/MDD가 없으므로 **운영 안정성, 로그, 데이터, 신호 생성** 기준으로 판단.

### SO-H1. 신호 생성 장기 실패 (Hard)
- 10영업일 연속 전 종목에서 신호 미생성 (ERROR/데이터 실패로 분석 자체가 안 된 경우)
- 정상적으로 HOLD가 나오는 것은 해당 안 됨 (시장 국면에 의한 정상 동작)
- **조치**: 데이터 소스/코드 확인 → 복구 후 재개

### SO-S1. 데이터 수집 간헐 실패 (Soft)
- 주간 기준 데이터 수집 실패 3회 이상 (같은 종목 반복 또는 다종목)
- **판단**: 특정 종목 문제 vs 전체 소스 문제 → 소스 전체 문제면 중단 검토

### SO-S2. 로그 이상 패턴 (Soft)
- 일간 ERROR 로그 10건 이상이 3일 연속
- **판단**: 코드 버그 vs 외부 원인 → 코드 문제면 실험 무효화 가능

### SO-S3. 신호 편향 (Soft)
- 20영업일 연속 전 종목 동일 신호 (예: 전부 HOLD)
- 시장 국면 bearish로 인한 정상 동작일 수 있음
- **판단**: 시장 국면 확인 → bearish면 관측 계속, bullish인데 전부 HOLD면 전략 로직 의심

## Full Paper 전용 Stop 조건

> Full paper는 P&L/포지션이 있으므로 **MDD, 일손실, 승률, 기대값, 거래 횟수** 기준으로 판단.

### FP-H1. MDD 25% 도달 (Hard)
- 누적 MDD가 25%에 도달하면 즉시 중단
- 참고: MDD 15% 시점에 시스템이 자동으로 신규 매수를 중단함 (관측 계속)
- **단계**: 15% → 시스템 자동 방어 (관측) → 20% → 경고 → **25% → 실험 중단**

### FP-H2. 일일 손실 5% 1회 발생 (Hard)
- 단일 일에 포트폴리오 손실 5% 초과
- 시스템 max_daily_loss 3% 방어를 돌파한 것 → 방어 로직 결함 의심
- **조치**: 즉시 중단, 방어 로직 점검

### FP-S1. 일일 손실 3% 3회 이상 (Soft)
- 실험 기간 중 일일 손실 3% 초과가 누적 3회 이상
- **판단**: 시장 급락 vs 전략 문제 → 시장 급락이면 계속, 전략 버그면 중단

### FP-S2. 전략 성과 열화 (Soft)
- `performance_degradation` 발동: 최근 20거래 승률 35% 미만
- 시스템이 자동으로 신규 매수를 중단함
- **판단**: 관측 계속 (자동 방어 동작 검증 목적), 20영업일 연속 발동 시 중단 검토

### FP-S3. 블랙스완 3회 이상 발동 (Soft)
- 실험 기간 중 블랙스완 감지가 3회 이상 발동
- **판단**: 각 발동 시점의 시장 상황 기록, 시스템 대응 정상 여부 확인

### FP-S4. Paper vs Backtest 괴리 (Soft)
- `paper_backtest_compare` 수익률 차이 15%p 초과, 또는 승률 차이 20%p 초과
- **판단**: 구현 버그 의심 → 원인 분석 (데이터 차이 vs 코드 문제)

### FP-S5. 누적 기대값 (EV) 음수 지속 (Soft)
- 30영업일 경과 후에도 거래당 기대값(평균 수익)이 음수
- 거래 건수가 10건 이상일 때만 판단 (샘플 부족 시 보류)
- **판단**: 전략 자체의 수익성 의심 → 중단 검토 (단, 비용 차감 전/후 구분 필요)

### FP-S6. 월간 거래 횟수 상한 초과 (Soft)
- `max_monthly_roundtrips: 8` 을 연속 2개월 이상 상한 도달
- 과매매(churning) 의심
- **판단**: 전략 조건이 빈번하게 바뀌는지 확인

## 중단하지 않는 경우 (관측만)

### Signal-Only
| 상황 | 이유 |
|------|------|
| 전 종목 HOLD가 계속됨 | 시장 국면 bearish면 정상 |
| 특정 종목 분석 1~2일 실패 | 간헐적 데이터 문제는 허용 |
| BUY 신호가 매우 드묾 | 엄격한 진입 조건 = 정상 |

### Full Paper
| 상황 | 이유 |
|------|------|
| 수익률이 마이너스 | 전략 관측이 목적, 손실도 데이터 |
| 벤치마크 하회 | 동일 |
| 거래가 장기간 없음 | 시장 국면(bearish) + 엄격 조건 = 정상 |
| 블랙스완 1~2회 발동 | 시스템 방어 동작 검증 |
| 특정 종목 손절 | 리스크 관리 동작 검증 |
| MDD 15% 도달 (자동 방어) | 관측 계속 (25%에서 중단) |
